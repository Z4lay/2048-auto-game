# 2048 自动游戏
## 项目简介
**2048 自动游戏** 是一个基于 **蒙特卡洛算法** 的自动化游戏代理。通过模拟多次游戏步骤，AI 能够在 2048 游戏中做出最优的决策，从而自动化玩游戏。这个项目演示了如何利用强化学习与搜索算法（如蒙特卡洛模拟）在经典益智游戏中实现智能决策。

## 功能
- 自动化玩 **2048** 游戏。
- 使用 **蒙特卡洛算法** 进行最优决策选择。
- 实现了基本的图形界面，用于显示游戏的每一步进程。
- 支持游戏结果的实时展示与保存。

## 安装与运行
### 1. 克隆项目
首先，克隆项目到本地：
```bash
git clone https://github.com/your-username/2048-auto-game.git
cd 2048-auto-game
```

### 2. 安装依赖
确保你已安装 Python 3，并使用 pip 安装以下依赖项：
```bash
pip install -r requirements.txt
```

### 3. 运行游戏
在终端中执行以下命令启动自动游戏代理：
```bash
python game.py
```

### 4. 自定义配置
你可以根据需要修改 `config.py` 文件中的参数，例如调整游戏的速度、决策算法的深度等。

## 代码结构
- `game.py`：主程序，负责游戏的初始化和代理的决策过程。
- `monte_carlo.py`：包含蒙特卡洛模拟算法，用于生成最优决策。
- `config.py`：存放游戏和算法的配置参数。
- `game_board.py`：实现 2048 游戏的棋盘逻辑与状态更新。

## 如何工作
1. **游戏初始化**：游戏会初始化一个 4x4 的棋盘，随机生成两个初始数字。
2. **代理决策**：通过蒙特卡洛算法，AI 会模拟每个可能的游戏动作，计算期望得分，并选择最优动作。
3. **游戏执行**：AI 根据选择的动作执行游戏，并实时更新棋盘状态。
4. **游戏结束**：当没有更多可移动的格子时，游戏结束。

## 项目发展计划
- **优化算法**：提升蒙特卡洛模拟的效率，减少计算时间。
- **强化学习**：引入强化学习算法（如 Q-Learning）进一步提高代理的表现。
- **多线程支持**：加入多线程功能以实现更高效的决策过程。
